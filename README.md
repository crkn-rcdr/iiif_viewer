## Installation
Add this line to your application's Gemfile:

```ruby
gem "iiif_viewer"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install iiif_viewer
```

Then, run the rake file that will add the required Javascript to your project.
```
rails iiif_viewer:cpfiles
```
